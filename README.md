## Backend Setup

### Install Dependencies

```bash
cd backend
npm install
```

### Run the Server (Development)

```bash
npm run dev
```

Starts the backend server with hot reloading for development.

### Run the Server (Production)

```bash
npm run build
npm run start
```

Builds the project and starts the production server.
